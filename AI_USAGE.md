# AI Usage Declaration

## Purpose

This document records the responsible use of Artificial Intelligence (AI) during the Software Development Life Cycle (SDLC) for this project. The objective is to maintain transparency regarding where AI-assisted tools were used while ensuring that all technical decisions and final deliverables remained under developer control.

---

## AI-Assisted Activities

AI tools were used only in the following areas:

### 1. Code Review Support

- Reviewed portions of the codebase to identify potential improvements.
- Suggested refactoring opportunities and coding best practices.
- Helped detect possible bugs and improve code readability.

### 2. Test Case Generation Support

- Assisted in generating positive, negative, and boundary test scenarios.
- Helped prepare API validation test cases.
- Suggested additional edge cases to improve test coverage.

### 3. CSS Implementation Assistance

- Assisted in implementing selected UI styling.
- Suggested responsive layout improvements.
- Recommended minor CSS optimizations for consistency and maintainability.

### 4. UI Improvement Suggestions

- Provided recommendations for improving user interface consistency.
- Suggested enhancements for spacing, alignment, typography, and user experience.
- Offered feedback on overall usability and visual presentation.

---

## Developer Responsibilities

The following activities were performed entirely by the developer:

- Project architecture and design
- Database schema design
- Feature planning and implementation
- Business logic development
- REST API implementation
- Security and authentication implementation
- Integration of frontend and backend
- Debugging and issue resolution
- Final code validation and testing
- Deployment configuration
- Performance optimization
- All implementation decisions and final code review

---

## Compliance

This project complies with the requirement to transparently document AI usage within the SDLC. AI assistance was limited to support activities and did not replace developer judgment or ownership of the final software deliverables.
